📌 Quick Reference — Jacobian Formula

J = ∂(u,v)/∂(x,y) = | ∂u/∂x ∂u/∂y | | ∂v/∂x ∂v/∂y | = (∂u/∂x)(∂v/∂y) − (∂u/∂y)(∂v/∂x)

Compute all four partial derivatives first, then evaluate the 2×2 determinant using the formula above.
